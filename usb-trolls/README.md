# Apple USB Flash Drive Trolls 🍎🔌

A collection of funny, harmless USB flash drive pranks for macOS and iOS users. All trolls are under 8GB and completely safe!

## Available Trolls

1. **Fake System Updates** - Makes it look like a critical system update is pending
2. **Infinite Folder Maze** - Creates a nested folder structure that's fun to navigate
3. **Sound Spam Generator** - Plays funny sounds at random intervals
4. **Fake Virus Scanner** - A fake antivirus that finds "threats" (which are just silly file names)
5. **Desktop Chaos Creator** - Auto-generates random files on desktop when plugged in
6. **Battery Drain Simulator** - Fake notification that battery is draining fast
7. **WiFi Password Faker** - Prompts for WiFi password with humorous messages

## Installation

1. Clone this repository
2. Navigate to the troll you want to use
3. Follow the specific instructions in each troll's README

## Requirements

- macOS 10.14+
- Python 3.6+
- USB Flash Drive (2GB minimum, 8GB maximum)

## Safety Warning ⚠️

All scripts in this collection are:
- ✅ Non-destructive (no data loss)
- ✅ Easily reversible
- ✅ Transparent about their pranking nature
- ✅ Respect user privacy

Always get permission before running these on someone else's device!

## Usage

Each troll folder contains its own executable script. See individual README files for specific instructions.

```bash
cd usb-trolls/[troll-name]
python3 install.py
```

Enjoy the pranks! 😄
